# SkillGod

Project memory and expert skills for this repository.

## Before coding
1. Prefer MCP tool sg_inject_context with the user task
2. Or run: sg inject "<task>"

## After coding decisions
1. Prefer MCP tool sg_capture_turn
2. Or run: sg capture --task "..." --output "..."
3. Or: sg remember "decision: ..."

Static memory also lives in AGENTS.md and SKILLGOD.md digests.
